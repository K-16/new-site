## Как скомпилировать?
* Устанавливаем [Node.JS](http://nodejs.org/download)
* Устанавливаем [Brunch](brunch.io): ```npm i -g brunch```
* ```npm i```
* ```brunch b```
* Также можно запустить слежение за файлами и сервер: ```brunch w -s```

## Деплой
* ```npm run deploy```
